# RiskGraph: Credit Score for Open-Source Packages

**Tagline:** One API call. Five risk signals. Know every package before installing.

**Description:**
RiskGraph analyzes npm and PyPI packages for risk signals in real-time — maintainer activity, version anomalies, download trends, license risk, and CVE history. Built for AI agents that autonomously install millions of dependencies.

**Key Features:**
- ✅ Real-time scoring against 5 risk signals
- ✅ Live CVE database via OSV.dev API
- ✅ npm + PyPI support (more ecosystems coming)
- ✅ REST API + CLI + Dashboard
- ✅ Open source (MIT)
